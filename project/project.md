기획의도 
- MBTI Universe는 성향기반으로 컨텐츠를 추천하고, 감정을 제어하며, 커뮤니티와
일정을 연결하는 통합 라이프스타일 플랫폼입니다. 다양한 기술 스택을 활용해 모듈화된 구조로
개발되었으며, 실무에서 요구되는 API 설계, 보안, UX데이터 분석까지 모두 담았습니다.
