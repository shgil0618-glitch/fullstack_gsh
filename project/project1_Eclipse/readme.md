# Project 1 - Eclipse 기반 Java 팀 프로젝트

## 프로젝트 개요

* 콘솔 환경에서 Java 기본 문법과 프로그램 구조를 학습하며 구현한 팀 프로젝트
* 레시피 관리 기능을 중심으로 입력 처리, 데이터 흐름, 예외 상황을 고려한 로직 설계 경험

## 프로젝트 정보

* **기간**: 2024.11 – 2024.12
* **구분**: 팀 프로젝트
* **개발 환경**: Eclipse
* **사용 기술**: Java

## 주요 기능

1. 레시피 등록 · 수정 · 삭제 · 조회 기능 구현
2. 사용자 입력 기반 메뉴 선택 및 흐름 제어
3. 조건문·반복문을 활용한 로직 처리
4. 예외 상황에 대한 기본적인 오류 처리

## 담당 업무 및 성과

* 레시피 관련 핵심 로직 설계 및 기능 구현
* 콘솔 입력값 검증 및 메뉴 이동 흐름 처리
* 데이터 구조 정리로 코드 가독성 및 유지보수성 향상

## 트러블슈팅

### 사례 1

* **문제**: 잘못된 사용자 입력으로 프로그램 흐름이 중단되는 문제 발생
* **원인**: 입력값에 대한 검증 로직과 예외 처리가 부족함
* **해결**: 입력값 범위 체크 및 예외 처리 로직 추가
* **성과**: 비정상 입력 상황에서도 안정적인 프로그램 실행 가능
* **학습**: 사용자 입력 검증과 예외 처리의 중요성 이해
* **향후 개선 계획**: 파일 저장 또는 DB 연동을 통한 데이터 영속성 강화

## 프로젝트 소감

* Java 기본 문법을 실제 기능 구현에 적용하며 프로그래밍 구조에 대한 이해가 향상됨
* 콘솔 기반이지만 사용자 흐름과 데이터 처리를 고려한 설계 경험이 이후 웹 프로젝트의 기초가 됨
* 팀 프로젝트를 통해 역할 분담과 코드 통합의 중요성을 체감함
