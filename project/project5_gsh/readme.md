

# 🍽️ 오늘 뭐먹지? – Recipe Sharing SNS

Spring Boot · JPA 기반 레시피 공유 **SNS형 UGC 웹 서비스**

---

## 1. 프로젝트 개요

**“오늘 뭐먹지?”**는 레시피 공유 서비스를 **SNS 구조로 확장**한
UGC(User Generated Content) 기반 웹 서비스입니다.

기존 게시판 중심 서비스에서 벗어나
**좋아요·리트윗·해시태그 등 SNS 인터랙션 요소**,
**JWT 기반 인증**, **Redis 캐싱**, **React UI**를 도입하여
실사용을 고려한 **확장성과 성능 중심 구조**를 목표로 개발했습니다.

* **프로젝트 성격**: 팀 프로젝트 (SNS형 서비스 설계·구현)
* **목표**: 트래픽·무결성·확장성을 고려한 백엔드 구조 경험

## 6. 트러블슈팅

### 사례 1. 좋아요 · 리트윗 중복 요청으로 인한 반응 수 비정상 증가

**문제**

* 동일 사용자가 같은 게시글에 좋아요·리트윗을 반복 요청하여
  반응 수가 비정상적으로 증가하는 문제 발생

**원인**

* 사용자–게시글 반응 상태에 대한 서버 단 중복 제어 부족
* 데이터 무결성 제약 조건 미흡

**해결**

* ERD 및 데이터 흐름 재검토
* 사용자 ID + 게시글 ID 기준 **유니크 제약 조건** 적용
* 서버 단 반응 토글 로직 구현

**성과**

* 반응 데이터 무결성 확보
* SNS 인터랙션 신뢰도 향상

**학습**

* SNS 서비스에서는 UI 제어뿐 아니라
  **데이터 레벨의 상태 관리가 필수적**임을 이해

**향후 개선 계획**

* 대규모 트래픽 환경 대비
* Redis 기반 반응 상태 캐싱 및 비동기 처리 도입 검토
