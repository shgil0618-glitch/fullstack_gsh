기술스택

  • Spring Boot 2.7.14: 안정적인 백엔드 프레임워크로, RESTful API 서버를 빠르게 구축하고 의존성 관리를 자동화합니다.
  • Spring Security 5.7.x: JWT 기반 인증 및 인가 구현을 위한 보안 프레임워크입니다.
  • Spring Data JPA 2.7.x: 객체와 관계형 데이터베이스 간의 매핑을 처리하며, CRUD 기능을 효율적으로 구현합니다.
  • Hibernate 5.6.x: JPA 구현체로, 복잡한 쿼리와 엔티티 관리를 지원합니다.
  • Oracle 11: 안정적인 관계형 데이터베이스로, 사용자 정보 및 게시글 데이터를 저장합니다.
  • React 18.x: 컴포넌트 기반의 프론트엔드 라이브러리로, 사용자 친화적인 UI와 SPA(Single Page Application)를 구현합니다.
  • Axios 1.x: 프론트엔드에서 백엔드 API와 비동기 통신을 수행하는 HTTP 클라이언트입니다.
  • React Router v6: 페이지 간 라우팅을 처리하여 동적인 화면 전환을 가능하게 합니다.
  • JWT (JSON Web Token): 사용자 인증 및 권한 관리를 위한 토큰 기반 인증 방식입니다.
  • Git & GitHub: 버전 관리 및 협업을 위한 도구로, 프로젝트의 변경 이력을 관리하고 배포 준비를 합니다.
  • GitHub Actions: CI/CD 자동화 도구로, 코드 변경 시 자동 테스트 및 배포 파이프라인을 구성합니다.
